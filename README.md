# Hotel Revenue Forecasting Dashboard

โปรเจกต์นี้สร้างแดชบอร์ดเพื่อพยากรณ์รายได้โรงแรม/ที่พักด้วยเวลาซีรีส์ และให้ผู้ใช้ปรับ "จำนวนผู้เข้าพัก" เพื่อจำลองสถานการณ์

---

## 🗂️ โครงสร้างไฟล์ (แต่ละไฟล์คืออะไรและใช้อย่างไร)

### 📄 `app.py`
- **ทำอะไร:** เป็นเว็บแอป Dash ที่แสดงกราฟรายได้จริงและพยากรณ์ พร้อมตัวเลือกภูมิภาค, ขนาดโรงแรม, และสไลเดอร์จำลองนักท่องเที่ยว
- **ใช้อย่างไร:** รัน `python app.py` แล้วเปิด `http://127.0.0.1:8050`

### 📁 `data/raw/datas.csv`
- **ทำอะไร:** ข้อมูลดิบ (Raw data) ที่รวบรวมจากแหล่งข้อมูลต้นทาง
- **ใช้อย่างไร:** ใช้เป็น input ในขั้นตอนการทำความสะอาดข้อมูล 

### 📁 `data/processed/datas_cleaned.csv`
- **ทำอะไร:** ข้อมูลที่ผ่านการทำความสะอาดและเตรียมสำหรับโมเดล
- **ใช้อย่างไร:** ใช้เป็น input ให้โน้ตบุ๊กหรือสคริปต์ฝึกโมเดล

### 📁 `data/predict/predictions.csv`
- **ทำอะไร:** ผลลัพธ์การพยากรณ์ที่ถูกนำมาใช้ในแดชบอร์ด
- **ใช้อย่างไร:** `app.py` จะโหลดไฟล์นี้เพื่อแสดงกราฟพยากรณ์ ถ้าปรับโมเดลให้สร้างไฟล์ใหม่ ให้บันทึกทับไฟล์นี้แล้วรีสตาร์ทแอป

### 📓 `notebooks/cleaning.ipynb`
- **ทำอะไร:** กระบวนการเตรียมข้อมูลและการเตรียม dataset สำหรับโมเดล
- **ใช้อย่างไร:** รันใน Jupyter Notebook เพื่อทำความสะอาดและบันทึกลง `data/processed/datas_cleaned.csv`

### 📓 `notebooks/modeling_timeseries_present_prediction.ipynb`
- **ทำอะไร:** สร้างโมเดลพยากรณ์สำหรับช่วงปัจจุบัน (ใน dataset) พร้อมการประเมินความแม่นยำ
- **ใช้อย่างไร:** รันโน้ตบุ๊กเพื่อตรวจสอบโมเดล, วิเคราะห์ความแม่นยำ, และปรับพารามิเตอร์

### 📓 `notebooks/modeling_timeseries_future_prediction.ipynb`
- **ทำอะไร:** สร้างโมเดลพยากรณ์ในอนาคต (forecast) และบันทึกไฟล์ `data/predict/predictions.csv`
- **ใช้อย่างไร:** รันโน้ตบุ๊กเพื่อเทรนโมเดลบนข้อมูลย้อนหลังและเขียนผลลัพธ์สำหรับแดชบอร์ด

#### 🔍 เปรียบเทียบสองโมเดล (Present vs Future)
- **โมเดล Present:** มุ่งเน้นที่การวัดความแม่นยำบนข้อมูลปัจจุบัน (การ validate โมเดลและ tuning) 
- **โมเดล Future:** มุ่งเน้นที่การ generate forecast สำหรับช่วงเวลาข้างหน้า (ผลลัพธ์ถูกนำไปใช้ในแดชบอร์ด)
- **ใช้งานจริง:** ให้ใช้ `present` เพื่อประเมินคุณภาพโมเดลก่อน แล้วใช้ `future` เพื่อสร้างไฟล์ `data/predict/predictions.csv` ที่จะถูกโหลดโดย `app.py`
